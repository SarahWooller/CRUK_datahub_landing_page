CRUK Filters

To help researchers find relevant datasets, the data hub can be filtered according to the cancers that it covers and the data types that it includes. It is therefore important that you categorise your dataset correctly. Please identify the cancers covered using the terms: Cancer ICD-O topography, Cancer ICD-O histology and CRUK cancer terms. These categories can be expanded by clicking on the words. Tick all the boxes that apply. Ticking a category will include all the sub-categories. Your dataset will be included in any query which includes any of the cancer types clicked. Now identify the data-types included using the terms: In Vitro Study, Animal Model, and Patient Study Again these categories can be expanded by clicking on the words. Your dataset will be included in all queries that are not looking for additional datatypes.
